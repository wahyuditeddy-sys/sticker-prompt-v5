🚀 Workflow Dasar — 4 Langkah
1

Pilih Platform Jual Dulu

Sebelum generate, tentukan platform penjualan. Setiap platform punya spesifikasi berbeda.

  • Etsy / Redbubble → stiker fisik / digital PNG, fokus kualitas tinggi
  • Shopee → pasar Indonesia, harga lebih kompetitif
  • LINE Creator → animasi/stiker chat, ukuran kecil
  • WhatsApp → pack 512×512px WebP, max 30 stiker/pack
  • DTF (Direct to Film) → cetak ke baju/merchandise, resolusi tinggi
2

Pilih Kategori Tema

Gunakan sidebar kiri untuk pilih tema stiker. Tips cepat:

  • Badge HIGH = permintaan pasar sedang tinggi → prioritaskan ini
  • Filter bulan (Jan–Dec) untuk tema seasonal / momen khusus
  • Filter Ever untuk tema yang laku sepanjang tahun (evergreen)
  • Kolom Search untuk cari tema spesifik (mis: "ramadan", "dino", "nurse")
3

Pilih Engine & Klik Generate

Pilih AI engine sesuai tool yang kamu pakai, lalu klik ✨ Generate Prompt:

  • Midjourney → kualitas terbaik, paste ke Discord /imagine
  • Gemini / Flow → Google tools, natural language, drag-drop langsung
  • NB2 ID → prompt Bahasa Indonesia terstruktur
  • EN Struct → prompt Bahasa Inggris terstruktur, cocok untuk referensi internasional
4

Copy Prompt → Paste ke AI Tool

Klik tombol Copy di prompt output, lalu paste ke tool AI yang kamu pilih.

  • Untuk 30 variasi sekaligus → klik ⚡ Generate 30 Vars
  • Untuk data listing toko → klik tab SEO Panel (judul, tags, deskripsi Etsy/Shopee)
  • Prompt sebelumnya tersimpan di tab History

🔧 Panduan Setiap AI Engine
🎨

Midjourney

Discord /imagine · niji 6 · v7
  • Kualitas visual terbaik untuk stiker
  • Paste prompt ke Discord bot: /imagine [prompt]
  • --niji 6 = gaya kawaii/anime
  • --v 7 = gaya realistis/ilustrasi
  • --stylize 850 = lebih kreatif & artistik
  • --ar 1:1 = square (single icon)
  • --ar 4:3 = landscape (sheet pack)
  • Klik U1–U4 untuk upscale setelah generate
  • Cocok untuk: Etsy, Redbubble, DTF
✨

Gemini Imagen

Google AI Studio · Gemini 2.0
  • Natural language, tidak perlu format khusus
  • Buka Google AI Studio → Gemini 2.0 Flash
  • Atau gunakan ImageFX dari Google
  • Prompt lebih panjang lebih baik
  • Bagus untuk: Shopee, karakter Sanrio-inspired
  • Gratis di Google AI Studio (dengan limit)
  • Cocok jika tidak punya akses Midjourney
🇮🇩

NB2 ID (Bahasa Indonesia)

Prompt terstruktur · Bahasa Indonesia
  • Output prompt dalam Bahasa Indonesia
  • Format terstruktur dengan seksi yang jelas
  • Cocok untuk tim/klien yang tidak fasih Inggris
  • Bisa paste ke AI lokal atau ChatGPT (DALL-E)
  • Terbaik untuk: Shopee, LINE, WhatsApp
  • Includ keyword SEO Bahasa Indonesia
🇬🇧

EN Struct (English Structured)

Prompt terstruktur · English · SEO-ready
  • Output prompt dalam Bahasa Inggris terstruktur
  • Sections: Category → Style → Format → Tech → Finish → Platform
  • Paling detail dan lengkap
  • Cocok untuk: Etsy, Redbubble, brief ke freelancer
  • Include negative prompt & platform specs
  • Terbaik untuk dokumentasi dan workflow tim
🌊

Flow (Stable Diffusion)

ComfyUI / A1111 · LoRA · CFG scale
  • Untuk pengguna Stable Diffusion
  • Include CFG scale, steps, sampler recommendations
  • Bisa run lokal (gratis, unlimited)
  • Butuh GPU yang cukup kuat (min RTX 3060)
  • Cocok untuk: DTF, eksperimen, volume besar
  • Include negative prompt detail (NoStickerLoRA)
🦁

Leonardo.ai

leonardo.ai · Anime Pastel Dream
  • Platform AI image khusus kreator
  • Buka leonardo.ai → Image Generation
  • Pilih model: Anime Pastel Dream untuk kawaii
  • Include model recommendation di prompt output
  • Cocok untuk: kawaii, anime-style, illustrasi
  • Gratis 150 token/hari (cukup untuk ~15 gambar)

📦 Spesifikasi Teknis Per Platform

🛍️ Etsy

  • Format: PNG (fisik) / SVG (digital)
  • Resolusi: min 300 DPI
  • Ukuran: min 3000×3000px untuk single
  • Background: transparan (PNG-24)
  • Ukuran file: max 20MB per file
  • Tips: upload 10+ listing foto preview
  • Terbaik: die-cut, holographic, premium

🫧 Redbubble

  • Format: PNG transparan
  • Resolusi ideal: 7632×6480px @ 72 DPI
  • Min: 1500×1200px
  • Background: harus transparan
  • Tips: upload resolusi max untuk print terbesar
  • Print-on-demand: Redbubble cetak dan kirim

🛒 Shopee

  • Format: JPG / PNG
  • Resolusi: min 800×800px
  • Rasio: 1:1 (square) untuk produk
  • Ukuran file: max 2MB per foto
  • Tips: buat mockup produk yang menarik
  • Digital: upload ZIP berisi PNG files

💬 LINE Creator Market

  • Format: PNG (transparan)
  • Ukuran main: 370×320px max
  • Tab icon: 96×74px
  • Chat icon: 240×240px
  • Max file: 1MB per stiker
  • Jumlah: 8, 16, atau 24 stiker/set
  • Proses review: 7-14 hari kerja LINE

📱 WhatsApp Sticker

  • Format: WebP (wajib)
  • Ukuran: 512×512px
  • Max file: 100KB per stiker
  • Jumlah: min 3, max 30 stiker/pack
  • Tool: WhatsApp Sticker Studio app
  • Tips: gunakan pngquant + cwebp untuk kompresi

👕 DTF (Direct to Film)

  • Format: PNG atau TIFF
  • Resolusi: min 150 DPI, ideal 300 DPI
  • Mode warna: CMYK (konversi dari RGB)
  • Ukuran: sesuai ukuran cetak (A3, A4)
  • Background: putih atau transparan
  • Tips: cek warna di layar vs cetak berbeda
